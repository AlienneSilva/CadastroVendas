"# CadastroVendas" 
"Cadastro de vendas
Requisitos funcionais do usuário;
1.	usuário deve ser capaz de realizar login;
2.	Usuário deve ser capaz de criar um cadastro básico com os dados da venda;
3.	Usuário deve ser capaz de salvar os dados;
4.	Usuário deve ser capaz de visualizar as suas vendas;
5.	Usuário deve ser capaz de atualizar dados do “card”;
Requisitos funcionais do sistema;
1.	Tela inicial de login e senha;
2.	Menu lateral com:
•	Cadastro;
•	Minhas vendas;
Tela de Cadastro:
•	Nome do cliente (empresa);
•	Nome do responsável;
•	Número de vidas;
•	Produto;
•	Cidade;
Tela de vendas
•	Número identificação do card;
•	Status;
"
